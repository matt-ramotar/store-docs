## Efficient data caching and retrieval

Optimize your app's performance and responsiveness with Store's powerful caching and data retrieval capabilities.

<CardGroup cols={3}>
  <Card
    title="In-Memory Caching for Quick Data Access"
    icon="head-side-brain"
    iconType="duotone"
    
  >
    Store provides in-memory caching backed by Guava, enabling efficient caching with minimal configuration.
  </Card>
  <Card
    title="Disk Caching for Offline Functionality"
    icon="floppy-disk"
    iconType="duotone"
  >
    Integrate effortlessly with databases like Room, SQLDelight, or Realm to persist data locally and provide a robust offline-first experience.
  </Card>

    <Card
    title="Customizable Caching Strategies"
    icon="gear"
    iconType="duotone"
    >
    Tailor your in-memory caching with configurable policies for TTL and maximum size. This ensures rapid data retrieval without unnecessary network calls.

  </Card>

</CardGroup>

## Reactive and reliable data streams

Build robust and responsive applications with Store's reactive data streaming and advanced error handling capabilities.

<CardGroup cols={3}>
  <Card
    title="Reactive Data Streams with Kotlin Flow"
    icon="wave"
    iconType="duotone"
    
  >
    Leverage Kotlin's Flow for a reactive API that emits streams of data, loading states, and errors. React to data changes in real-time from memory cache, disk cache, or network sources, keeping your UI consistently up-to-date.
  </Card>
  
  <Card
    title="Structured Error Handling Without Exceptions"
    icon="shield-halved"
    iconType="duotone"
  >
    Store wraps errors instead of throwing exceptions, keeping your data streams alive. Continue receiving updates even after encountering errors, promoting a more resilient and fault-tolerant application architecture.
  </Card>

    <Card
    title="Support for Observable Queries"
    icon="eyes"
    iconType="duotone"
    >
    Integrate with databases like Room to support observable queries. Store emits new data whenever the underlying database changes, even if the changes occur outside of Store.

  </Card>

</CardGroup>

## Advanced features for modern development

Enhance your app with Store's advanced capabilities designed to optimize network usage and streamline data management.

<CardGroup cols={3}>
  <Card
    title="Throttling and Inflight Request Deduplication"
    icon="traffic-light-stop"
    iconType="duotone"
    
  >
    Prevent redundant network calls by throttling API requests when multiple parallel requests are made for the same resource. The inflight debouncer ensures identical requests receive the same response, optimizing network usage and improving performance.
  </Card>
  
  <Card
    title="Full CRUD Operations with MutableStore"
    icon="pencil"
    iconType="duotone"
  >
    Extend Store's capabilities with MutableStore to support create, read, update, and delete operations. Manage both local and network data comprehensively, simplifying data handling and synchronization.
  </Card>

    <Card
    title="Built for Multiplatform Projects"
    icon="rocket"
    iconType="duotone"
    >
    Designed with multiplatform support, Store is suitable for common, JVM, Native, and JS projects. use Store across various platforms with consistent behavior, making it a versatile choice for diverse development environments.

  </Card>

</CardGroup>
